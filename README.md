## MONAWE PRJ 소개 및 선정 이유
 - 소개 <br>
  . 국내 문구 Commerce 사이트 [모나미](https://www.monamimall.com/w/main.do) 클론 프로젝트<br>(짧은 기간, 개발에만 집중하기 위해서 디자인/기획 부분만 클론하였습니다.)
